# AL-KO XML Proxy для Rozetka

Автоматический прокси-сервис для трансформации XML прайс-листа AL-KO под требования маркетплейса Rozetka.

## Що робить цей скрипт?

Автоматично виправляє всі проблеми, які вказав менеджер Rozetka:

| Проблема | Рішення |
|----------|---------|
| 1. Товари в наявності з кількістю 0 | Автоматично ставить `available="false"` якщо `stock_quantity <= 0` |
| 2. Товари без фото | Виключаються з прайс-листа |
| 3. Посилання в параметрах | Автоматично видаляються |
| 4. Різні написання виробника | Нормалізується до "AL-KO" |
| 5. Бренд в назві товару | Додається після типу товару (Газонокосарка AL-KO Moweo...) |
| 6. Дублікати назв | Робляться унікальними через додавання артикулу |
| 7. Недостатньо параметрів | Можна налаштувати додаткові в конфігурації |

## Встановлення

### Крок 1: Завантаження файлів

Завантажте всі файли на ваш хостинг в окрему папку, наприклад:
```
/public_html/rozetka-feed/
```

Структура файлів:
```
rozetka-feed/
├── index.php          # Основний скрипт
├── config.php         # Налаштування
├── cache/             # Папка для кешу (створіть вручну)
│   └── .gitkeep
└── logs/              # Папка для логів (створіть вручну)
    └── .gitkeep
```

### Крок 2: Налаштування прав доступу

```bash
chmod 755 index.php
chmod 644 config.php
chmod 777 cache/
chmod 777 logs/
```

### Крок 3: Редагування конфігурації

Відкрийте `config.php` і перевірте налаштування:

```php
return [
    // Ваше джерело XML
    'source_xml_url' => 'https://apipim.al-ko.ua/storage/xml_files/PriceList.xml',
    
    // Інформація про магазин для Rozetka
    'shop' => [
        'name' => 'AL-KO Technics Ukraine',
        'company' => 'alko-technics.com.ua',
        'url' => 'https://alko-technics.com.ua',
    ],
    
    // Товари для виключення (без фото тощо)
    'exclude_articles' => [
        '123097',
        '127192',
    ],
    
    // Час кешування (секунди)
    'cache' => [
        'enabled' => true,
        'time' => 3600, // 1 година
    ],
];
```

### Крок 4: Перевірка

Відкрийте в браузері:
```
https://alko-technics.com.ua/rozetka-feed/
```

Ви повинні побачити XML з виправленим прайс-листом.

## Посилання для Rozetka

Після встановлення надайте менеджеру Rozetka це посилання:

```
https://alko-technics.com.ua/rozetka-feed/
```

або

```
https://alko-technics.com.ua/rozetka-feed/index.php
```

## Додаткові налаштування

### Виключення товарів

Якщо потрібно виключити певні товари (наприклад, без фото або ассортиментні):

```php
'exclude_articles' => [
    '123097',  // Товар без фото
    '127192',  // Товар без фото
    // Додайте інші артикули
],
```

### Видалення непотрібних параметрів

```php
'remove_params' => [
    'Ставка ПДВ',
    'Кількість вантажних місць',
    // Інші параметри для видалення
],
```

### Налаштування кешування

Для зменшення навантаження на сервер-джерело:

```php
'cache' => [
    'enabled' => true,
    'time' => 3600, // 1 година = 3600 секунд
],
```

Збільште час кешування якщо прайс-лист оновлюється рідко.

## Перевірка прайс-листа

Після налаштування перевірте прайс-лист валідатором Rozetka:
https://seller.rozetka.com.ua/gomer/pricevalidate/check/index

## Логування

Логи зберігаються в папці `logs/` у форматі:
```
proxy_2025-12-04.log
```

Перевіряйте логи якщо виникають проблеми.

## Усунення проблем

### Помилка "Не удалось загрузить XML"
- Перевірте доступність джерела: https://apipim.al-ko.ua/storage/xml_files/PriceList.xml
- Перевірте налаштування firewall на хостингу

### Помилка "Ошибка парсинга XML"
- Можливо джерело XML має помилки синтаксису
- Перевірте логи для детальної інформації

### Пустий результат
- Перевірте права доступу до папок cache/ та logs/
- Подивіться логи помилок PHP

## Автоматичне оновлення

XML оновлюється автоматично при кожному запиті з урахуванням часу кешування.

Якщо потрібне примусове оновлення - видаліть файл кешу:
```bash
rm cache/pricelist_cache.xml
```

## Контакти

При виникненні питань звертайтесь до розробника.
