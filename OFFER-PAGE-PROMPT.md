## Технічні вимоги

### Фреймворк та компоненти
- **React** — нативні React-компоненти (JSX/TSX)
- Кожна секція — окремий компонент
- Використовувати `useState`, `useEffect`, `useRef` де потрібно (анімації при скролі, тощо)
- Компонентна архітектура: `<Hero />`, `<PaymentsStrip />`, `<ProblemsSection />`, `<FeaturesSection />`, `<ComparisonTable />`, `<PricingSection />`, `<AiBotSection />`, `<ProcessSection />`, `<GuaranteesSection />`, `<CtaSection />`
- Стилізація: **Tailwind CSS** або CSS Modules — на вибір, але єдиний підхід на весь проект
- Можна використовувати Framer Motion для анімацій (scroll-triggered animations, fade-in, slide-up)

### Іконки
- Бібліотека: **lucide-react** (`npm install lucide-react`)
- Приклад: `import { ShoppingBag, Truck, CreditCard } from 'lucide-react'`
- НЕ використовувати емоджі як іконки

### Шрифт
- Google Fonts: **Plus Jakarta Sans** (wght 400-800)
- Fallback: -apple-system, BlinkMacSystemFont, sans-serif

### Сітка
- Всі блоки з картками — рівномірна сітка
- Три колонки на десктопі (`grid-cols-3` або `grid-template-columns: repeat(3, 1fr)`)
- Дві колонки на планшеті
- Одна колонка на мобільному
- Однакова кількість елементів у кожному рядку

- Чистий, сучасний, без перевантаження
- Картки з тонкою рамкою (#e2e8f0), border-radius: 20px
